## 项目所用技术

* VueQuillEditor：富文本编辑器
* ECharts：图表库
* VueParticles：粒子特效
* Vue2 + Vuex + Vue Router + Element UI + ES6 + webpack + axios + Node.js

[后端传送](https://github.com/scrpio/springboot-master)
